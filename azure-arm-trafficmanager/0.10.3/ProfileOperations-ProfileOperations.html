ProfileOperations

Operations for managing WATMv2 profiles. NOTE: An instance of this class is automatically created for an instance of the [TrafficManagerManagementClient] TrafficManagerManagementClient~TrafficManagerManagementClient. See [profiles] TrafficManagerManagementClient~TrafficManagerManagementClient#profiles. Initializes a new instance of the ProfileOperations class.
