Class: AddOnOperations

AddOnOperations

Provides REST operations for working with Store add-ins from the Windows Azure store service. NOTE: An instance of this class is automatically created for an instance of the [StoreManagementClient] StoreManagementClient~StoreManagementClient. See [addOns] StoreManagementClient~StoreManagementClient#addOns. Initializes a new instance of the AddOnOperations class.
