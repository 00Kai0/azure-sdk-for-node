NetworkSecurityGroupOperations

The Network Resource Provider API includes operations for managing the NetworkSecurityGroups for your subscription. NOTE: An instance of this class is automatically created for an instance of the [NetworkResourceProviderClient] NetworkResourceProviderClient~NetworkResourceProviderClient. See [networkSecurityGroups] NetworkResourceProviderClient~NetworkResourceProviderClient#networkSecurityGroups. Initializes a new instance of the NetworkSecurityGroupOperations class.
