ApplicationGatewayOperations

ApplicationGatewayOperations

The Network Resource Provider API includes operations managing the application gateways for your subscription. NOTE: An instance of this class is automatically created for an instance of the [NetworkResourceProviderClient] NetworkResourceProviderClient~NetworkResourceProviderClient. See [applicationGateways] NetworkResourceProviderClient~NetworkResourceProviderClient#applicationGateways. Initializes a new instance of the ApplicationGatewayOperations class.
