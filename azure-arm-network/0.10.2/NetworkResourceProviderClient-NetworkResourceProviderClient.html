NetworkResourceProviderClient

The Windows Azure Network management API provides a RESTful set of web services that interact with Windows Azure Networks service to manage your network resrources. The API has entities that capture the relationship between an end user and the Windows Azure Networks service. Initializes a new instance of the NetworkResourceProviderClient class.
