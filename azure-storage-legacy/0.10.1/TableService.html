Class: TableService

TableService

The TableService object allows you to peform management operations with the Microsoft Azure Table Service. The Table Service stores data in rows of key-value pairs. A table is composed of multiple rows, and each row contains key-value pairs. There is no schema, so each row in a table may store a different set of keys.

For more information on the Table Service, as well as task focused information on using it from a Node.js application, see How to Use the Table Service from Node.js.
