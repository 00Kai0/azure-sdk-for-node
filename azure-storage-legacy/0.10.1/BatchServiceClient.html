Class: BatchServiceClient

BatchServiceClient

new BatchServiceClient()

Creates a new BatchServiceClient.

Implements a batch service client able to produce OData batch requests from an array of operations. For more information about OData batch processing refer to http://www.odata.org/developers/protocols/batch.
