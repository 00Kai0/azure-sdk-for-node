Class: FirewallRuleOperations

FirewallRuleOperations

The Azure SQL Database Management API includes operations for managing the server-level Firewall Rules for Azure SQL Database Servers. You cannot manage the database-level firewall rules using the Azure SQL Database Management API; they can only be managed by running the Transact-SQL statements against the master or individual user databases. NOTE: An instance of this class is automatically created for an instance of the [SqlManagementClient] SqlManagementClient~SqlManagementClient. See [firewallRules] SqlManagementClient~SqlManagementClient#firewallRules. Initializes a new instance of the FirewallRuleOperations class.
