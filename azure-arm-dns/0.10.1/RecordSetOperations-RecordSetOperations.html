Class: RecordSetOperations

RecordSetOperations

Operations for managing the RecordSets in a DNS zone. NOTE: An instance of this class is automatically created for an instance of the [DnsManagementClient] DnsManagementClient~DnsManagementClient. See [recordSets] DnsManagementClient~DnsManagementClient#recordSets. Initializes a new instance of the RecordSetOperations class.
