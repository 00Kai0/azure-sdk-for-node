Class: DNSServerOperations

DNSServerOperations

The Compute Management API includes operations for managing the dns servers for your subscription. NOTE: An instance of this class is automatically created for an instance of the [ComputeManagementClient] ComputeManagementClient~ComputeManagementClient. See [dnsServer] ComputeManagementClient~ComputeManagementClient#dnsServer. Initializes a new instance of the DNSServerOperations class.
