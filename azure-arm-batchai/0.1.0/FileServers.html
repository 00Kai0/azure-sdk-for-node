Name of the resource group to which the
resource belongs.
fileServerName string

The name of the file server within the
specified resource group. File server names can only contain a combination
of alphanumeric characters along with dash (-) and underscore (_). The name
must be from 1 through 64 characters long.
parameters object

The parameters to provide for file server
creation.

Properties
Name Type Attributes Description

The region in which to create the File

the File Server.

The size of the virtual machine of the
file server. For information about available VM sizes for fileservers from
the Virtual Machines Marketplace, see Sizes for Virtual Machines (Linux).
sshConfiguration object

SSH settings for the file
server.

Properties
Name Type Attributes Description

List of source
IP ranges to allow SSH connection to VM. Default value is '*' can be used to
match all source IPs. Maximum number of publicIPs that can be specified are
400.
userAccountSettings object

Settings for
user account of VMs.

Properties
Name Type Attributes Description
adminUserName string

Specifies the

adminUserSshPublicKey string <optional>

returned in a GET response body.
adminUserPassword string <optional>

Admin

body.

Settings for the data disk which would
be created for the file server.

Name Type Description

Initial disk size in GB

disks.
diskCount number

Number of data disks to be

storageAccountType string

Specifies the type
