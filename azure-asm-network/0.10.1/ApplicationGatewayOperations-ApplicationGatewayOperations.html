ApplicationGatewayOperations

The Application Gateway Management API includes operations for managing application gateways in your subscription. (see http://msdn.microsoft.com/en-us/library/windowsazure/jj154113.aspx for more information) NOTE: An instance of this class is automatically created for an instance of the [NetworkManagementClient] NetworkManagementClient~NetworkManagementClient. See [applicationGateways] NetworkManagementClient~NetworkManagementClient#applicationGateways. Initializes a new instance of the ApplicationGatewayOperations class.
