Class: NetworkSecurityGroupOperations

NetworkSecurityGroupOperations

The Network Management API includes operations for managing the Network Security Groups for your subscription. NOTE: An instance of this class is automatically created for an instance of the [NetworkManagementClient] NetworkManagementClient~NetworkManagementClient. See [networkSecurityGroups] NetworkManagementClient~NetworkManagementClient#networkSecurityGroups. Initializes a new instance of the NetworkSecurityGroupOperations class.
