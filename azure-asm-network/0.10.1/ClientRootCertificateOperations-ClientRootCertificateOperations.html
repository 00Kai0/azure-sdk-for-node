ClientRootCertificateOperations

The Network Management API includes operations for managing the client root certificates for your subscription. (see http://msdn.microsoft.com/en-us/library/windowsazure/jj154113.aspx for more information) NOTE: An instance of this class is automatically created for an instance of the [NetworkManagementClient] NetworkManagementClient~NetworkManagementClient. See [clientRootCertificates] NetworkManagementClient~NetworkManagementClient#clientRootCertificates. Initializes a new instance of the ClientRootCertificateOperations class.
