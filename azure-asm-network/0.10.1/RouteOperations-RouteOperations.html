Class: RouteOperations

RouteOperations

The Network Management API includes operations for managing the routes for your subscription. NOTE: An instance of this class is automatically created for an instance of the [NetworkManagementClient] NetworkManagementClient~NetworkManagementClient. See [routes] NetworkManagementClient~NetworkManagementClient#routes. Initializes a new instance of the RouteOperations class.
