sendBadge(tags, value, optionsOrCallback, callback) → {undefined}

Sends a wns/badge WNS notification.

Parameters:
Name Type Argument Description
tags array | string

Comma-separated list or array of tag identifiers.
value object | string

Either a numeric value or a string value that specifies a predefined badge glyph. Numerically, this value can accept any valid integer. A value of 0 clears the badge, values from 1-99 display as given, and any value greater than 99 displays as 99+. See http://msdn.microsoft.com/en-us/library/windowsazure/br212849.aspx for more details.
