AvailabilitySetOperations

Operations for managing the availability sets in compute management. NOTE: An instance of this class is automatically created for an instance of the [ComputeManagementClient] ComputeManagementClient~ComputeManagementClient. See [availabilitySets] ComputeManagementClient~ComputeManagementClient#availabilitySets. Initializes a new instance of the AvailabilitySetOperations class.
