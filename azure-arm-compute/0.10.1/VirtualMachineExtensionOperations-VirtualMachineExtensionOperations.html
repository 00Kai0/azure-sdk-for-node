Class: VirtualMachineExtensionOperations

VirtualMachineExtensionOperations

Operations for managing the virtual machine extensions in compute management. NOTE: An instance of this class is automatically created for an instance of the [ComputeManagementClient] ComputeManagementClient~ComputeManagementClient. See [virtualMachineExtensions] ComputeManagementClient~ComputeManagementClient#virtualMachineExtensions. Initializes a new instance of the VirtualMachineExtensionOperations class.
