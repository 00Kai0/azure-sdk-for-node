VirtualMachineImageOperations

Operations for managing the virtual machine images in compute management. NOTE: An instance of this class is automatically created for an instance of the [ComputeManagementClient] ComputeManagementClient~ComputeManagementClient. See [virtualMachineImages] ComputeManagementClient~ComputeManagementClient#virtualMachineImages. Initializes a new instance of the VirtualMachineImageOperations class.
