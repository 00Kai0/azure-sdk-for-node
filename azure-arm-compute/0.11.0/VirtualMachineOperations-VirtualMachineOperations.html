VirtualMachineOperations

VirtualMachineOperations

Operations for managing the virtual machines in compute management. NOTE: An instance of this class is automatically created for an instance of the [ComputeManagementClient] ComputeManagementClient~ComputeManagementClient. See [virtualMachines] ComputeManagementClient~ComputeManagementClient#virtualMachines. Initializes a new instance of the VirtualMachineOperations class.
