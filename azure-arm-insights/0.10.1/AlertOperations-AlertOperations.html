AlertOperations

NOTE: An instance of this class is automatically created for an instance of the [InsightsManagementClient] InsightsManagementClient~InsightsManagementClient. See [alertOperations] InsightsManagementClient~InsightsManagementClient#alertOperations. Initializes a new instance of the AlertOperations class.
