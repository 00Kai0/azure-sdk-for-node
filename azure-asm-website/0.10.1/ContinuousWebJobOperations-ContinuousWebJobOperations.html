ContinuousWebJobOperations

Operations for managing continuous WebJobs. NOTE: An instance of this class is automatically created for an instance of the [WebSiteExtensionsClient] WebSiteExtensionsClient~WebSiteExtensionsClient. See [continuousWebJobs] WebSiteExtensionsClient~WebSiteExtensionsClient#continuousWebJobs. Initializes a new instance of the ContinuousWebJobOperations class.
