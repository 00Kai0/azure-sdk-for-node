WebSiteManagementClient

The Web Sites Management API provides a RESTful set of web services that interact with the Windows Azure Web Sites service to manage your web sites. The API has entities that capture the relationship between an end user and Windows Azure Web Sites service. (see http://msdn.microsoft.com/en-us/library/windowsazure/dn166981.aspx for more information) Initializes a new instance of the WebSiteManagementClient class.

getOperationStatus(webSpaceName, siteName, operationId, callback) → {Stream}

The Get Operation Status operation returns the status of the specified operation. After calling a long-running operation, you can call Get Operation Status to determine whether the operation has succeeded, failed, timed out, or is still in progress. (see http://msdn.microsoft.com/en-us/library/windowsazure/ee460783.aspx for more information)
