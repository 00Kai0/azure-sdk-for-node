Class: WebsiteManagementService

WebsiteManagementService

The WebsiteManagementService object allows you to perform management operations on Microsoft Azure Web Sites.

new WebsiteManagementService(configOrSubscriptionId, authentication, hostOptions)

Creates a new WebsiteManagementService object.

Parameters:
Name Type Argument Description
configOrSubscriptionId string

Configuration information, subscription ID for the account or the connection string
authentication object

The authentication object for the client. You must use either keyfile/certfile or keyvalue/certvalue to provide a management certificate to authenticate service requests.

Properties
Name Type Argument Description
keyfile string <optional>

The path to a file that contains the PEM encoded key
certfile string <optional>

The path to a file that contains the PEM encoded certificate
keyvalue string <optional>

A string that contains the PEM encoded key
certvalue string <optional>

A string that contains the PEM encoded certificate
hostOptions object <optional>

The host options to override defaults.

Properties
Name Type Argument Default Description
host string <optional>
 'management.core.windows.net'

The management endpoint.
apiversion string <optional>
 '2012-03-01'

The API vesion to be used.
