WebSpaceOperations

Operations for managing web spaces beneath your subscription. NOTE: An instance of this class is automatically created for an instance of the [WebSiteManagementClient] WebSiteManagementClient~WebSiteManagementClient. See [webSpaces] WebSiteManagementClient~WebSiteManagementClient#webSpaces. Initializes a new instance of the WebSpaceOperations class.
