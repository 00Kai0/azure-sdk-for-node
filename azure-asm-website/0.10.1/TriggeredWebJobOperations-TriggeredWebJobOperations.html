TriggeredWebJobOperations

Operations for managing Triggered WebJobs. NOTE: An instance of this class is automatically created for an instance of the [WebSiteExtensionsClient] WebSiteExtensionsClient~WebSiteExtensionsClient. See [triggeredWebJobs] WebSiteExtensionsClient~WebSiteExtensionsClient#triggeredWebJobs. Initializes a new instance of the TriggeredWebJobOperations class.
