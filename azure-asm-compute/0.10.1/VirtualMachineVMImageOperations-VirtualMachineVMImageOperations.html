Class: VirtualMachineVMImageOperations

VirtualMachineVMImageOperations

The Service Management API includes operations for managing the virtual machine templates in your subscription. NOTE: An instance of this class is automatically created for an instance of the [ComputeManagementClient] ComputeManagementClient~ComputeManagementClient. See [virtualMachineVMImages] ComputeManagementClient~ComputeManagementClient#virtualMachineVMImages. Initializes a new instance of the VirtualMachineVMImageOperations class.
