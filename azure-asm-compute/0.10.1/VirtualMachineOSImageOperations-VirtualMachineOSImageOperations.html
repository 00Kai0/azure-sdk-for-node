VirtualMachineOSImageOperations

The Service Management API includes operations for managing the OS images in your subscription. (see http://msdn.microsoft.com/en-us/library/windowsazure/jj157175.aspx for more information) NOTE: An instance of this class is automatically created for an instance of the [ComputeManagementClient] ComputeManagementClient~ComputeManagementClient. See [virtualMachineOSImages] ComputeManagementClient~ComputeManagementClient#virtualMachineOSImages. Initializes a new instance of the VirtualMachineOSImageOperations class.

Unreplicate an OS image to multiple target locations. This operation is only for publishers. You have to be registered as image publisher with Windows Azure to be able to call this. Note: The operation removes the published copies of the user OS Image. It does not remove the actual user OS Image. create(parameters, callback) → {Stream}

The Create OS Image operation adds an operating system image that is stored in a storage account and is available from the image repository. (see http://msdn.microsoft.com/en-us/library/windowsazure/jj157192.aspx for more information)

Parameters:
Name Type Description
parameters VirtualMachineOSImageCreateParameters

Parameters supplied to the Create Virtual Machine Image operation.

Properties
Name Type Argument Description
label string

Specifies the friendly name of the image.
mediaLinkUri string

Specifies the location of the blob in Azure storage. The blob location must belong to a storage account in the subscription specified by the SubscriptionId value in the operation call. Example: http://example.blob.core.windows.net/disks/mydisk.vhd.
name string

Specifies a name that Azure uses to identify the image when creating one or more virtual machines.
operatingSystemType string

The operating system type of the OS image. Possible values are: Linux or Windows.
eula string <optional>

Specifies the End User License Agreement that is associated with the image. The value for this element is a string, but it is recommended that the value be a URL that points to a EULA.
description string <optional>

Specifies the description of the OS image.
imageFamily string <optional>

Specifies a value that can be used to group OS images.
publishedDate date <optional>

Specifies the date when the OS image was added to the image repository.
isPremium boolean

Indicates if the image contains software or associated services that will incur charges above the core price for the virtual machine.
showInGui boolean

Specifies whether the image should appear in the image gallery.
privacyUri string <optional>

Specifies the URI that points to a document that contains the privacy policy related to the OS image.
iconUri string <optional>

Specifies the URI to the icon that is displayed for the image in the Management Portal.
recommendedVMSize string <optional>

Specifies the size to use for the virtual machine that is created from the OS image.
smallIconUri string <optional>

Specifies the URI to the small icon that is displayed when the image is presented in the Azure Management Portal. The SmallIconUri element is only available using version 2013-03-01 or higher.
language string <optional>

Specifies the language of the image. The Language element is only available using version 2013-03-01 or higher.
