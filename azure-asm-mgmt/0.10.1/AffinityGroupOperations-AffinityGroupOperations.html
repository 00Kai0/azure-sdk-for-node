A name for the affinity group that is unique to the subscription.

A base-64-encoded name for the affinity group. The name can be up to 100 characters in length. Its value defaults to the Name of the affinity group if not provided.
description string <optional>

A description for the affinity group. The description can be up to 1024 characters in length.
