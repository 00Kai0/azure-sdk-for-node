Class: ManagementCertificateOperations

ManagementCertificateOperations

You can use management certificates, which are also known as subscription certificates, to authenticate clients attempting to connect to resources associated with your Azure subscription. (see http://msdn.microsoft.com/en-us/library/windowsazure/jj154124.aspx for more information) NOTE: An instance of this class is automatically created for an instance of the [ManagementClient] ManagementClient~ManagementClient. See [managementCertificates] ManagementClient~ManagementClient#managementCertificates. Initializes a new instance of the ManagementCertificateOperations class.
